Books API Documentation

GET /books

Supported Methods: GET, HEAD

Query Parameters: None

Body Parameters: None

Returns: JSON object containing all books in memory.

Example Response:

{
  "statusName": "success",
  "message": "Books retrieved successfully.",
  "books": [
    {
      "id": 1,
      "title": "Things Fall Apart",
      "author": "Chinua Achebe",
      "year": 1958,
      "country": "Nigeria",
      "language": "English",
      "link": "https://en.wikipedia.org/wiki/Things_Fall_Apart",
      "pages": 209,
      "genres": ["Historical Fiction", "Postcolonial Literature"]
    }
  ]
}

GET /books/search

Supported Methods: GET, HEAD

Query Parameters:

ParamTypeRequiredDescription
authorstringoptionalFilter books by author name (case-insensitive, partial match)
titlestringoptionalFilter books by title (case-insensitive, partial match)
yearnumberoptionalFilter books by exact publication year
genrestringoptionalFilter books by genre (partial or full match)
limitnumberoptionalMaximum number of results to return

Body Parameters: None

Returns: JSON object containing filtered book results.

Example Response:

{
  "statusName": "success",
  "message": "Filtered results.",
  "results": [
    {
      "title": "Things Fall Apart",
      "author": "Chinua Achebe",
      "year": 1958,
      "country": "Nigeria",
      "language": "English",
      "link": "https://en.wikipedia.org/wiki/Things_Fall_Apart",
      "pages": 209,
      "genres": ["Historical Fiction", "Postcolonial Literature"]
    }
  ]
}

POST /books

Supported Methods: POST

Body Parameters (JSON):

ParamTypeRequiredDescription
titlestringrequiredTitle of the book
authorstringrequiredAuthor of the book
yearnumberoptionalPublication year
genresarray of stringsoptionalBook genres

Returns: JSON object containing the newly created book.

Example Response:

{
  "statusName": "created",
  "message": "Book added successfully.",
  "book": {
    "title": "New Book",
    "author": "Author Name",
    "year": 2025,
    "genre": ["genre1", "genre2", ...]
  }
}

POST /books/:id

Supported Methods: POST

Body Parameters (JSON): Any book fields to update (title, author, year, genres, etc.)

Returns: JSON object containing the updated book

Errors: 404 if book not found, 400 if invalid JSON
